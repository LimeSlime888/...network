# ...network
this repository was created for Our World of Text's [...network](https://owot.me/...network).
the ...network chat is a shared chat between many worlds that aims to be a moderated alternative of the global chat.
...network's canvas centre of interaction is [...world](https://owot.me/...network). it aims to be a moderated alternative of the front page.

## channels
channels function like tags on a message. non-goatway bots MUST use the 'bot' channel.
one can blacklist or whitelist channels. to see the channels of a message, one can double-click it.

## moderation
the [moderators](/moderators.txt) moderate according to the [rules](/rules.md). moderators can:
* limit in chat by:
  * muting (m)
  * ratelimiting (l)
  * force-channeling (c)
  * force-unchanneling (C)
* limit specific users/anons
* limit all anons
* limit everyone except moderators
* delete messages:
  * by referring to their position relative to the last message
  * by triple clicking a message
* access the Time Machine of /...world and /...network
* be member on /...world and /...network
  * write 20480 characters/s
  * member-protect areas

it also enables anyone to:
* delete messages locally if they're not a moderator
* filter channels
