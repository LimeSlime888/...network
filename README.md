# ...network
created for Our World of Text's [...network](https://owot.me/...network)
## chat.js features
chat.js enables the moderation filter for ...network's chat. it allows moderators to:
* mute/ratelimit specific users/anons
* mute/ratelimit all anons
* mute/ratelimit everyone except moderators
* delete messages:
  * by referring to their position relative to the last message
  * by clicking while holding the Ctrl key
